Ex10TimeHelpers¶

The Ex10TimeHelpers are an abstraction that the reader needs to implement for various ways get the current time. This timer needs to implement a timer with at least millisecond accuracy.

struct Ex10TimeHelpers¶

Helpers for working with board specific timing.

This API should be used when timing is needed from the host rather than the ex10 device.

Public Members

uint32_t (*time_now)(void)¶

Grabs the current time to mark as the start of a timer.

Return

uint32_t The number of milliseconds elapsed since the start of the program.

uint32_t (*time_elapsed)(uint32_t start_time)¶

Returns the time elapsed since the start_time parameter.

Return

uint32_t The number of milliseconds elapsed since start_time.

void (*busy_wait_ms)(uint32_t msec_to_wait)¶

Wait for a specific number of milliseconds.

Param msec_to_wait

The number of milliseconds to wait.

void (*wait_ms)(uint32_t msec_to_wait)¶

Suspends execution for the specified number of milliseconds.

Param msec_to_wait

The number of milliseconds to wait, will assert if > 1000ms.
