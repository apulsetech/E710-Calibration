Ex10 Reader PrimitivesÂ¶

The Ex10 reader primitives are a set of modules in the Ex10 SDK that is a collection of tools that the use cases can assemble as needed for a particular task. These primitives are built as individual modules so that they may or may not be included in the application. This allows for better management of code size in applications that are sensitive to that.
