Impinj Reader Chip SDK ExamplesÂ¶

The Impinj reader chip SDK comes with examples that have three different types. The first type is the Use Case Examples which show how to implement different reader functionality using the use case layers. Then the second is the legacy examples that use the Ex10Reader to implement different reader functionality. The third are miscellaneous examples that show how to do non inventory related operations. The use case examples are the preferred method for new development as it allows for more flexibility in the reader implementation.
