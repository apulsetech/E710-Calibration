Ex10Calibration¶

The Ex10Calibration reader primitive is used to convert the requested transmit power in cdBm to and from ADC counts that will be used with the Impinj reader chips internal power meters. This calibration is implementation dependent so it relies on the calibration implementation in the boards directory. This also provides conversion for the RSSI counts to and from cdMm for the tag read packets, and the LBT ops.

The customers can customize the calibration system. For an in depth look at refer to Calibration Customization Using CalGen in the advanced topics documentation
