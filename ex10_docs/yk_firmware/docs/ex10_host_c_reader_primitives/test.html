Ex10Test¶

The Ex10Test reader primitive is not used in normal Impinj Reader chip operations such as inventory or access commands. This is used for elements that are more related to development such as just ramping up the transmitter to CW. Or running the PRBS test op that will output PRBS data on the transmitter that is useful for spectral mask testing.
