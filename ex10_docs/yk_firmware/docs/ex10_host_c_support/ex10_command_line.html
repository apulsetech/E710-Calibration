Ex10CommandLine¶

The Ex10CommandLine module is a helper function that is used to parse command line arguments for the use case examples. This allows the user to be able to modify the behavior of the use case examples depending on which arguments they support.

struct Ex10Result ex10_parse_command_line(char const *const *argv, int argc, struct Ex10CommandLineArgument const *arguments, size_t arguments_size, union Ex10CommandLineArgumentValue *values)¶

Parse the command line arguments from main().

Each node in the arguments array specifies how a command line argument is to be parsed into this array. If a command line specifier is encountered, then the associated node in this array is set to the parsed value. If no value is found which matches the specifier, then the default value is used (i.e. the value element within the values array is unchanged).

Parameters

  • argv – The argument list passed into main.

  • argc – The number of arguments to parse.

  • arguments – The specifiers and container of parsed arguments.

  • arguments_size – The number of nodes in the parameter arguments.

  • [in – out] values The parsed values array. The number of nodes in this array must match the arguments_size parameter.
