Ex10Random¶

The Ex10Random struct is used to abstract away the implementation of the random number generator. The E710 Development Kit board uses the srand() and rand() functions of the standard C library. In a more embedded environment there could be a different method for generating a random number. These random functions are used in the examples and for shuffling the the region channels when required (FCC).
