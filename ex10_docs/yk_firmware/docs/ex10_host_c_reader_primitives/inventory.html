Ex10Inventory¶

The Ex10Inventory reader primitive provides an interface to run an inventory round. This is a primitive inventory round from the Impinj reader chip’s point of view. The use case layer will use this to start an inventory round after it has ramped up power in a channel. The use cases will also use this module to start up inventory if the dynamic Q algorithm has finished.

This module will run the SendSelectOp if it is enabled, and then run the StartInventoryRoundOp to start the inventory round, and then return. If the send selects option is enabled, it assumes that the application/use case has already configured the correct registers for the SendSelectOp

struct Ex10Result (*start_inventory)(uint8_t antenna, enum RfModes rf_mode, int16_t tx_power_cdbm, struct InventoryRoundControlFields const *inventory_config, struct InventoryRoundControl_2Fields const *inventory_config_2, bool send_selects)¶

Starts an Inventory by running the StartInventoryRoundOp.

This function calls Ex10Inventory.run_inventory() after performing the preconditions:

  • Ramp up the transmitter, if necessary.

  • Update the temperature measurement.

  • Set the RF mode.
